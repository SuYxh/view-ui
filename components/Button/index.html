Button 按钮 ​

常用的操作按钮。

基础用法 ​

使用 type、plain、round 和 circle 来定义按钮的样式。

<template>
  <div class="button-wrapper">
    <x-button> hello </x-button>
    <x-button type="primary"> Primary </x-button>
    <x-button type="danger"> Danger </x-button>
    <x-button loading> Loading </x-button>
  </div>
</template>

<style scoped>
.button-wrapper {
  display: flex;
  gap: 10px;
}
</style>
